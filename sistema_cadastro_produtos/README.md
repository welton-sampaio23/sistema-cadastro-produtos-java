# Sistema de Cadastro de Produtos (Java)

Projeto em Java desenvolvido para praticar:
- Programação Orientada a Objetos (POO)
- Herança
- Listas (ArrayList)
- Validação de dados
- CRUD em console

## Funcionalidades
- Cadastrar produtos alimentos e eletrônicos
- Buscar produtos
- Atualizar preço
- Remover produtos
- Calcular valor total em estoque

## Tecnologias
- Java
- Eclipse
